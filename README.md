Descripcion del programa:<br>
-El programa del repositorio cuenta con una estructura modular y bien comentada para garantizar una buena comprension del mismo.<br>
-Su funcionalidad es gestionar informacion sobre paises, los cuales se encuentran en el archivo paises.csv<br>

Participación de los integrantes:<br>
-Franco Gagliardi: Desarrollo de las funciones para mostrar y modificar los datos del archivo CSV.<br>
-Ramiro Quiroga: Implementación de las funciones para filtrar y ordenar los datos.<br>
-Trabajo conjunto: Ambos colaboraron en la construcción del main, integrando todas las funcionalidades en un menú interactivo.<br>

Instrucciones de uso:<br>
-Ejecutar el archivo main.py.<br>
-Seleccionar una opción del menú:<br>
-Mostrar países<br>
-Modificar datos<br>
-Filtrar por continente o población<br>
-Ordenar por nombre o población<br>
-Seguir las instrucciones en pantalla para completar la acción.<br>

Ejemplo de entrada:<br>

Ingrese una opción:<br>
1 - Mostrar países<br>
2 - Modificar país<br>
3 - Filtrar países<br>
4 - Ordenar países<br>
5 - Salir<br>

Si el usuario elige 2, se le pedirá:<br>

Ingrese el nombre del país a modificar:<br>
> Argentina<br>
Ingrese el nuevo valor de población:<br>
> 47000000<br>

Ejemplo de salida:<br>

País modificado correctamente.<br>
Argentina ahora tiene una población de 47.000.000 habitantes.
